Приветствую!

Для запуска дашборда в среде linux локально, необходимы библиотеки:

dash
dash_core_components
dash_html_components

datetime

matplotlib

plotly

pandas

json

Пожалуйста! Установите недостающие библиотеки командой в консоли:
pip3 install <библиотека>

Перейдите в консольном режиме в папку  cod_of_win.
Выполните команду:
python3 teddy_dash.py

Откройте ссылку http://127.0.0.1:8050/

Должен запуститься дашборд.
